# Promolocation PWA

This folder contains the separate browser-first app for the Promolocation migration.

## Scope of this scaffold

- Standalone web entry point
- Shared imports from `@promolocation/shared`
- No native Expo map, geofencing, push, or biometric dependencies

## Planned next steps

1. Add the auth shell and session storage strategy.
2. Add the live browser geolocation hook.
3. Build the activation map route with a browser-safe map library.
4. Migrate incidents, reporting, profile, and QR flows.
